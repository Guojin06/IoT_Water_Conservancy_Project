# 🚀 阿里云通义千问配置指南

## 🎯 快速获取API Key

### 1. 访问DashScope控制台
打开：https://dashscope.aliyuncs.com/

### 2. 登录/注册
- 使用阿里云账号登录
- 或者新注册账号

### 3. 获取API Key
1. 进入控制台
2. 点击"API-KEY管理"
3. 创建新的API Key
4. 复制生成的API Key

### 4. 更新配置
将API Key更新到 `config.py`：
```python
ALIBABA_API_KEY = "sk-xxxxxxxxxx"  # 您的API Key
```
