## 🚀 快速启动

### 方法1：一键启动（推荐）
```bash
双击运行: START_SYSTEM.bat
```

### 方法2：手动启动
```bash
# 1. 测试C++系统
test_system.exe

# 2. 启动Web服务
python simple_server.py

# 3. 启动AI服务（可选）
cd ai_service
ai_env\Scripts\activate
python app.py
```
